Welcome to the official WaterPaths documentation!

WaterPaths is a C++ package for the optimization of multi-actor water distribution systems under deep uncertainty. It is a utility planning and management tool built to facilitate the identification of regionally-robust water supply planning and management policy pathways. To view its associated publication, please see Trindade et al, 2020.

Install

WaterPaths can be installed by following the instructions in the GitHub Repository that contains the original implementation of the code (Trindade et al, 2020).
