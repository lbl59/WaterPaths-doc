Welcome to the official WaterPaths documentation!

WaterPaths is a C++ package for the optimization of multi-actor water distribution systems under deep uncertainty. It is a utility planning and management tool built to facilitate the identification of regionally-robust water supply planning and management policy pathways. For the original implementation of the code, please see the GitHub Repository <https://github.com/bernardoct/WaterPaths> here.
