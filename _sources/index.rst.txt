Welcome to the official WaterPaths documentation!
=================================================

.. epigraph:: 
   **WaterPaths** is a C++ package for the optimization of multi-actor water distribution systems under deep uncertainty. It is a utility planning and management tool built to facilitate the identification of regionally-robust
   water supply planning and management policy pathways. For the original implementation of the code, please see the `GitHub Repository <https://github.com/bernardoct/WaterPaths>` here.

Install
=======
WaterPaths can be installed by following the instructions in the `GitHub Repository <https://github.com/bernardoct/WaterPaths>` that contains the original implementation of the code (Trindade et al, 2020).
